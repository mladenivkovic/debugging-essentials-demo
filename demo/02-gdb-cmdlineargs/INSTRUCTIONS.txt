02 - Demo Command Line Arguments
=====================================

1	  Look through the source code of the program.

2   Compile and Run the program:

    $ make
    $ ./ex

2.1 It crashes if we don't provide a command line argument, like so:

    $ ./ex some_argument some_other_argument

3   Run program with gdb without --args: It will still crash.

    $ gdb ./ex 
    (gdb) run

    $ gdb ./ex 1 2 3 4
    (gdb) run

4   Now Telling gdb that our program takes cmdline args:
4.1 Run program inside gdb with `run --args a b c`

    $ gdb ./ex 
    (gdb) run --args 1 2 3 4

    This works!

4.2 run program with gdb --args

    $ gdb --args ./ex 1 2 3 4 
    (gdb) run 
    
    This works too!
